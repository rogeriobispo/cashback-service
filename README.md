# README

## Motivation
  Market place to share promotions with cashback

## Addons
    * authentication to admin area
    * Javascript Counting down date 
    
## Requiriments
      * Ruby Version: 2.6.0
      * Rails 6.0.2.2
      * PostgreSQL 10.12


## Setup
    * git clone git@github.com:rogeriobispo/cashback-service.git
    * bundle install
    * rails db:create
    * rails db:migrate
    * rails db:seed
    * rails s
